# Order Discount Demo

This small project calculates the final amount after applying a percentage discount.

## Business rules

- `subtotal` must be a finite number greater than or equal to `0`.
- `discountPercent` must be a finite number between `0` and `100` inclusive.
- The returned monetary amount must be rounded to two decimal places.
- Invalid input must produce a clear error instead of a negative or non-finite amount.
