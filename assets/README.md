# Tribe Assets

部落资产是成员在任务中可以发现、评估和采用的外部资源。资产本身不等于已安装依赖。

每项资产包含：

- `asset.yaml`：身份、来源、状态、适用场景和风险。
- `blueprint.md`：官方资料整理出的采用图纸。
- `experience.md`：部落成员亲自验证后的经验，只记录可追踪 Run；没有验证时保持为空。

资产状态：

- `candidate`：已发现，尚未由部落实测。
- `trial`：正在隔离任务中验证。
- `adopted`：通过验收，可供成员选择使用。
- `deprecated`：不再推荐，但保留历史。

成员使用资产前必须检查状态、适用条件和图纸。`candidate` 资产不能被描述为部落已有经验。
